# ESP32-P4-Function-EV-Board

The ESP32-P4-Function-EV-Board is a multimedia development board based on the ESP32-P4 chip. The ESP32-P4 features a dual-core RISC-V processor and supports up to 32 MB of PSRAM. In addition, the ESP32-P4 supports various peripherals such as USB 2.0 standard, MIPI-CSI/DSI, and H264 Encoder, meeting customer needs for developing low-cost, high-performance, and low-power multimedia products.
