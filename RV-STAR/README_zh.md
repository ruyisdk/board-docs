# Nuclei RV-STAR

Nuclei RV-STAR 是一款基于 GD32VF103 MCU 的 RISC-V 开发板，搭载运行频率为 108MHz 的 Nuclei Bumblebee 内核。它配备 128KB Flash 和 32KB SRAM，为嵌入式开发提供了充足的资源。该板集成了板载调试器 (FTDI) 和 JTAG 接口，支持灵活的调试选项。它提供用于下载、调试和串口通信的 USB Type-C 接口，以及用于 USB OTG 功能的 Micro USB 接口。扩展能力包括兼容 Arduino 的接口、PMOD 接口（I2C、SPI）和 EXMC 内存扩展接口。该板还配备用户按键（Reset、Wakeup）、RGB LED 和一个默认未焊接的 microSD 卡槽。
