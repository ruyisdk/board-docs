# Nuclei RV-STAR

Nuclei RV-STAR is a RISC-V development board based on the GD32VF103 MCU, featuring the Nuclei Bumblebee core running at 108MHz. It comes with 128KB Flash and 32KB SRAM, providing sufficient resources for embedded development. The board integrates an on-board debugger (FTDI) and JTAG interface, supporting flexible debugging options. It offers USB Type-C for download, debug, and serial communication, along with a Micro USB port for USB OTG functionality. Expansion capabilities include Arduino-compatible interfaces, PMOD interfaces (I2C, SPI), and EXMC memory expansion interface. The board also features user buttons (Reset, Wakeup), an RGB LED, and a microSD card slot (unpopulated by default).
