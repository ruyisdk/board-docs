# EBC7700

EBC7700 是一款基于 ESWIN EIC7700X SoC 的高性能 RISC-V 边缘计算开发板。EIC7700X 搭载四核 SiFive Performance P550 64 位 RISC-V 处理器，支持 RV64GC 指令集，并集成了算力高达 20 TOPS (INT8) 的自研神经网络计算单元 (NPU)。该开发板板载高速 LPDDR5 内存，拥有丰富的音视频处理能力与外设接口，可完美运行 Ubuntu 24.04 等 Linux 系统。
