# EBC7700

The EBC7700 is a high-performance RISC-V edge computing development board based on the ESWIN EIC7700X SoC. The EIC7700X features a quad-core SiFive Performance P550 64-bit RISC-V processor supporting the RV64GC instruction set, and integrates a self-developed Neural Network Processing Unit (NPU) with up to 20 TOPS (INT8) of computing power.

The board is equipped with high-speed LPDDR5 memory and provides rich peripheral interfaces along with advanced audio and video processing capabilities. It supports running Linux-based operating systems such as Ubuntu 24.04.
