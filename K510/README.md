# Canaan K510-CRB-V1.2 KIT

The Canaan K510-CRB-V1.2 KIT is a developer kit based on the K510 chip. It adopts a hierarchical design, built around the K510 core module, with rich peripherals for easy expansion. It supports high-speed communication interfaces such as USB 2.0 OTG+PHY, Gigabit Ethernet, 3×SDIO, and 3×SPI, as well as low-speed communication interfaces including 4×UART, GPIO, I2C, and I2S. Easy to use and with abundant documentation, it helps customers quickly achieve concept evaluation and product prototype verification, shortening the product design cycle.
